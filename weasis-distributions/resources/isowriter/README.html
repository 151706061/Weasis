DICOM CD/DVD/USB media created by Weasis

This media contains medical images that can be read by a DICOM viewer like Weasis.

  1. Download Weasis
  2. Install Weasis
  3. Read the media by clicking on the CD-ROM button in the menu bar.

Layout of CD/DVD/USB media

  • DICOMDIR

    The DICOMDIR file (medical directory) is a DICOM encoded file containing summary information and references to all images on the CD.

  • DICOM

    The DICOM directory contains images that can be read by a DICOM viewer.

  • JPEG (Optional)

    This folder contains images converted to JPEG that can be read by any image viewer. The conversion of these images does not preserve the original quality of the DICOM files and does not preserve the metadata.

  • README

    The README file is an HTML file containing some basic information.
